    <details>
    <summary><b>рЯУЛ Heroku Setup</b></summary>

    ```bash
    heroku create your-bot-name
    heroku buildpacks:set heroku/nodejs
    git push heroku main
    heroku ps:scale worker=1
    ```
    > Uses a `worker` dyno вАФ the bot is a persistent process, not a web server.
    </details>

    <details>
    <summary><b>рЯУЛ Railway Setup</b></summary>

    ```bash
    railway init
    railway up
    ```
    Set the start command to `node index.js` in your Railway project settings.
    </details>

    <details>
    <summary><b>рЯУЛ Render Setup</b></summary>

    1. Fork this repo
    2. Connect your fork to Render вЖТ **New Background Worker**
    3. Set start command: `node index.js`
    4. Add your `.env` variables in the **Environment** tab
    5. Click **Deploy** вАФ the bot connects automatically

    </details>
